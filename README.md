# 🌊 SeisMark

Proof of Seismic Contribution

SeisMark is a simple Web3 tool that allows users to mint their profile picture as an NFT on the Seismic testnet.

## Features
- PFP to NFT
- Contributor badge
- Simple UI

## How it works
1. Enter your X username
2. Load your PFP
3. Mint your SeisMark
